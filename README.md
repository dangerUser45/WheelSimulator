# Wheel Simulator

## Что уже сделано
  На текущий момент в проекте реализовано следующее:

  - настроена сборка проекта через `CMake`;
  - подключены основные библиотеки: `OpenGL`, `GLFW`, `glad`, `Dear ImGui`,
  `ImPlot`, `stb_image`;
  - добавлены `GLM` и `Eigen3` как база для дальнейших вычислений и
  математической части;
  - реализовано создание окна приложения и инициализация
  `OpenGL`-контекста;
  - настроен основной цикл приложения;
  - подключён интерфейс на `Dear ImGui` с поддержкой docking;
  - добавлены панели управления и отображения статистики;
  - реализован тестовый график в реальном времени через `ImPlot`;
  - подготовлена основа для подключения текстур и внешних ресурсов.

  Таким образом, уже готов каркас приложения, на который можно опираться
  при дальнейшей разработке симуляции.

  ## Что планируется сделать
  С учётом того, что до завершения проекта осталось примерно **3 недели**,
  дальнейшая работа будет сосредоточена на доведении проекта до состояния
  **минимально жизнеспособной демонстрационной версии (MVP)**.

  В ближайшее время планируется:

  - заменить тестовые данные на данные, получаемые из модели симуляции;
  - реализовать базовую физическую модель движения колеса;
  - связать вычисляемые параметры с интерфейсом;
  - добавить отображение ключевых характеристик симуляции, например
  скорости, угла поворота и других параметров;
  - по возможности добавить визуальное представление колеса или сцены;
  - провести отладку, исправить ошибки и подготовить проект к демонстрации.

  ## План на оставшиеся 3 недели
  Текущий план работ можно сформулировать так:

  - **1 неделя** — реализация минимальной логики симуляции и подключение её
  к интерфейсу;
  - **2 неделя** — доработка визуализации и отображения результатов;
  - **3 неделя** — тестирование, исправление ошибок и подготовка финальной
  демонстрации.

## Сборка
Основные зависимости выставлены в `conan`.
`stb_image` лежит в репозитории в `3rdparty/stb`.

### Linux
```bash
git clone https://github.com/dangerUser45/WheelSimulator.git
cd WheelSimulator

conan install . -of .cache/conan -s build_type=Release --build=missing
cmake -S . -B build \
  -DCMAKE_BUILD_TYPE=Release \
  -DSANITIZE=OFF \
  -DCMAKE_TOOLCHAIN_FILE=.cache/conan/conan_toolchain.cmake
cmake --build build
```

Запуск:

```bash
./build/WheelSimulator
```
